# INSTA-PLUS-PLUS
PROJECT
•	A C++ Application developed with the help of OOP and Data Structures like Maps, Vector, Strings, etc. which imitates functionalities of very famous social media application Instagram.
•	Users can login/ signup by providing the username and password. Username validation is also implemented.
•	After login/ sign up, users will have the options like Edit profile (username, password, bio), search accounts/ follow accounts, check following/ followers, check posts (newest to oldest) / create posts , delete account/ logout.
